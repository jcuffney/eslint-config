# eslint-config

My opinionated ESLint configs for JavaScript/Node projects.

#### 1. Install
```shell
$ npm install --save-dev @cuffney/eslint-config
```

#### 2. Extend
```yaml
# .eslintrc
extends: @cuffney/eslint-config/node
```

#### Configs

| config       | extends              | with                                 |
|--------------|----------------------|--------------------------------------|
| `node`       | `eslint:recommended` |  `node`                              |
